# AI in Healthcare: Diabetes Prediction

## 📌 Overview
This project is a machine learning-based system that predicts the likelihood of diabetes using patient health data such as glucose level, BMI, age, and blood pressure.

## 🚀 Features
- Predicts diabetes using ML algorithms
- Data preprocessing and analysis
- User-friendly output
- Helps in early disease detection

## 🛠️ Technologies Used
- Java / Python (jo bhi use kiya ho)
- Machine Learning
- Pandas, NumPy (if used)

## 📊 How it Works
The model is trained on healthcare data and uses classification algorithms to predict whether a person is diabetic or not.

## ▶️ How to Run
1. Clone the repository
2. Run the project file
3. Input patient data
4. Get prediction result
